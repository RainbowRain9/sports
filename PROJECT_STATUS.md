## 🎯 项目描述

体育赛事管理系统是一个基于 **Vue.js + Element UI + Egg.js + MySQL** 技术栈开发的现代化体育赛事管理平台。系统支持完整的四角色工作流程，为管理员、操作员、运动员、裁判员提供专门的功能模块，采用现代化UI设计和流畅的交互体验。
